# Quiz Interativo - React

Projeto de Quiz interativo simples e responsivo.
A aplicação permite que o usuário responda perguntas sequenciais, veja sua pontuação e reinicie o quiz para uma nova tentativa.

## Funcionalidades

- Tela inicial possui mensagem de boas-vindas e botão para dar início ao quiz
- Perguntas exibidas uma de cada vez, com 4 alternativas cada.
- Feedback visual ao selecionar uma alternativa.
- Contador de acertos atualizado de acordo com as respostas.
- Botão "Próxima" para avançar entre as perguntas.
- Tela final exibindo o resultado do quiz.
- Opção de reiniciar.

## Tecnologias Utilizadas

- HTML – Estrutura da aplicação (index.html)
- CSS – Estilização, responsividade e animações (style.css)
- JavaScript – Lógica das perguntas, pontuação e interação com o DOM (script.js)
